# plusコマンド
![test](https://github.com/aka1125/robosys2025/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す。

# 使用例

```bash
$ git clone...
$ cd robosys2025
$ seq 3 | ./plus
$ 6
```

# インストール手順

```bash
$ git clone https://github.com/aka1125/robosys2025
$ cd robosys2025
$ - chmod +x plus
```

## 必要なソフトウェア
- Python
  - テスト済みバージョン: 3.7〜3.10

## テスト環境
- Ubuntu 22.04.5 LTS
- Python 3.12(64bit)
